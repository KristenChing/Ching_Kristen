Basic Stuff

First, in order to create a website, you have to define a file as an HTML. In the basics of HTML code, all commands are within <>, which are called tags. In order to call a command, you would have to put it within tags and to close it, put it within tags but with a slash after the opening tag. (Which ends up sounding really complicated because I can't type an example without it being interpreted as code.) So to define a file as an HTML, you type < ! doctype html >.

The basic commands are html/body/head, which define the parts of the website. The head contains "hidden" content and the body contains the main content which will be seen by the viewer. You can also add a title, which is the tab name.

Print formatting

HTML does not show returns/tabs and will print stuff all in one line. Which is why the p 'command' exists, because it is for creating a new paragraph! This is used in the same way as the title. As a result, each sentence appears on a new line. However, you can also use /br to create linebreaks. Additional ways to edit text are strong (bold) and em (italicize). Font sizes have a range from h1 to h5, h1 being the largest and h5 being the smallest.

Other Commands

- comments are added using an exclamation point followed by two dashes on both sides inside a pair of tags.
- a horizontal line is created using hr/.
- links are added using a (which stands for 'anchor'), followed by href =, then the link address and link name. Here is a link to a website.
- creating section titles: font size, followed by name =, identifier name, followed by actual name. The identifier name is used so it can be linked to from other links.
- linking to other sections: a, href =, "# identifier name"
- adding images img src = "imagename" height = 0 width = 0
- link titles: a href = "sitename" title = "link title" link name
