Basic Stuff

First, in order to create a website, you have to define a file as an HTML. In the basics of HTML code, all commands are within <>, which are called tags. In order to call a command, you would have to put it within tags and to close it, put it within tags but with a slashafter the opening tag. So to define a file as an HTML, you type < ! doctype html >.

The basic commands are < html >, < body >, and < head >, which define the parts of the website. The head contains "hidden" content and the body contains the main content which will be seen by the viewer. You can also add a title, which is the tab name.

Print formatting

HTML does not show returns/tabs and will print stuff all in one line. Which is why the < p > exists, because it is for creating a new paragraph! You can also use < /br > to create linebreaks. Additional ways to edit text are < strong > (bold) and < em > (italicize). Font sizes have a range from < h1 > to < h5 >, < h1 > being the largest and < h5 > being the smallest.

Other Commands

  • comments look like this: < !--this is a comment-- >
  • a horizontal line: < hr/ >.
  • links are added using < a href = link address >a link< /a > Here is a link to a website.
  • creating section titles: The identifier name is used so it can be linked to from other links.
  • linking to other sections: < a href = #"identifier name" >name< /a >
  • adding images: < img src = "imagename" height = 0 width = 0 >
  • link titles: < a href = "site address" title = "link title" >link name< /a >

Tables

Basically to create a table you write < table >. Inside the table you can have table rows, < tr >, and inside each < tr > you have < td >, which is table data. A table border is created using < table border = "num" >, which creates a border around the table, and 1 px lines dividing values. We can also add images and links inside of our table.

We can create headers for our table by using < th >, which stands for table header, inside of our < td > 's. We can also have an empty cell just by typing < td >< /td > The table header is automatically bolded so we don't have to bold it. In addition, we can add a title for the whole table, which is done inside a table header tag: < th colspan = "num" >Title< /th > We can adjust the height and width of our table using percentages so that it will take up that portion of the table. This can be done inside of a th tag or inside of the entire table tag. Another thing that is useful is cellpadding (cellpadding = ""). This is the space between the text and the cell border (pixels), and is set inside of the table tag. Lastly, cellspacing (cellspacing = "") adjusts the thickness of the borders that separate the cells.

CSS

Basically cool font stuff. Colors, font, etc. < style type = "text/css" > <-- activates CSS. MUST be in the head. ex. p {color:blue;font-family:arial} - this changes paragraph text so that the words in the paragraph are blue and in arial font. You can also just use RGB values. This works the same way for editing spacing, such as line-height:0% Whatever you are editing, in this case, p , the paragraph text, is called the element. The property is what you are changing about the element text. (color, font, etc.) You can even edit the entire body element.

Furthermore, you can edit more than one type at a time simply by using a comma. :O CSS properties:

  • {line-height:num%;} (spacing)
  • {font-weight: bold;}
  • {font-style:italic;}
  • {font-family:arial}

  • {text-indent:0px;}
  • {background-color:blue;}(this must be done in the body in order to edit background of the webpage, otherwise it will just create a "highlight" over the text)
  • {background-image:url(nameoffile.png);} ("tiles" itself in background)
  • {background-repeat:no-repeat;} (and then it will "no tile")
  • {background-repeat:repeat-x;} (row)

  • {background-position: 0px 2px;} (x y from top left corner/can also be done with percentages)

Div, Span and Classes

The Div tag: can be used to completely surround an amount of text. it is used to separate different sections.
The Span tag: only the text between the opening and closing span tags is affected, basically used to isolate parts of text
To create a class, you define it in the style tag and use it as the element. This is done by adding a . before the class name. The class is linked to the text by typing in the opening tag (which can be anything) class = "classname"

Style Sheets, pseudo elements, child selectors, and ID

a style sheet is used to apply consistent css effects to multiple webpages without having to copy and paste everything. The style sheet is on a separate css file and contains the default styling information. In order for the style sheet to take effect, the style sheet is linked to the html file in the head. < link rel = "stylesheet" type = "text/css" href = "stylesheet.css" > Any part of the style sheet can be overridden at any time by using the style tags we are already using.
pseudo elements are cool effects that are added on to an element, such as :first-letter{} :after{} :before{}
Child selectors allow distinction between parts of text, esp. if you want to edit one paragraph of text, but don't want to edit everything that is in that font. div > p means that div is the parent and p is the child. so it will edit the p that is inside of the div.
lastly, ID tags. These are created by putting id = "" into the starting/opening tag ie < p id = "none" > and accessed in css by #none{}

Positioning and Sizing

{position: static(default)/absolute/relative/fixed} (ex. text surrounded by border ) static is the default, and follows the page formatting. the border will stretch across the rest of the line. a second text will appear below it. In absolute both will appear surrounded only to edges of the text on top of each other on the top left corner. Relative positioning it is positioned relative to its original position. In fixed positioning, it doesn't move. It stays where it is.
{max-height:} / {max-width:} controls max height and max width without changing proportions of images
